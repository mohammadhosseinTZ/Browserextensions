## how does it work ??

### By clicking on the extension icon, you can personalize the website
<img width="1120" alt="nav" src="https://github.com/mohammadhosseinTZ/chromeExtensions/assets/146634977/78b034f3-fa0d-4d14-ac9c-d88e8cddccc6">

## By using this NAVBAR you can:
# 1)
Use the search box to filter the items on the page by the word you entered.
# 2)
By clicking on the "Remove Advertising" checkbox, remove all ads on the page.
# 3)
You can specify how many pages to see together on one page.
# 4)
And finally, you can specify what categories to see or even not see!

# Other features:

<img width="1118" alt="another" src="https://github.com/mohammadhosseinTZ/chromeExtensions/assets/146634977/9b34caf2-c7a6-45f7-98bc-ee618c9ee4a2">
With the stepper on the right side of the page, you can see how many pages you see on this page, and the height of the lines between the pages indicates the approximate number of items on each page, by clicking on each page number, the page scrolls to that page.
Also, the number of items on each page can be seen in the box above the items.

<img width="1112" alt="pagination" src="https://github.com/mohammadhosseinTZ/chromeExtensions/assets/146634977/758c5357-042d-4662-8a03-4a7a583c7394">
At the end of each page, there is a pagination that changes the appearance of this pagination with the customizations you have made, such as the number of pages you see on one page.
